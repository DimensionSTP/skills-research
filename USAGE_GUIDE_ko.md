# 사용 가이드

## 전체 흐름

1. `AUTONOMOUS_RESEARCH_SYSTEM.md`와 `RUN_PLANNING.txt`로 기획을 수행합니다.
2. `OUTPUT_CONTRACT.md`에 정의된 산출물을 생성합니다.
3. 실험을 수행하고 결과 템플릿을 채웁니다.
4. `RESULT_TO_PAPER_ENGINE.md`와 `RUN_WRITING.txt`로 초안을 작성합니다.

## 레포 내 템플릿 산출물

- `experiment_design.md`
- `result_schema.json`
- `research_blueprint.json`
- `experiment_results.json`
- `paper_seed.md`
- `appendix_next_runs.md`

위 파일은 시작용 템플릿이며 `SAMPLE` 내용을 실제 값으로 교체해야 합니다.

## 검증 팁

- blueprint와 results의 실험 ID를 일치시킵니다.
- 실험 시작 후에는 schema 키를 안정적으로 유지합니다.
- 정보가 부족하면 가정을 명시적으로 기록합니다.
