# Research Workflow Cheatsheet

## Planning

- Input: idea paragraph
- Use: `RUN_PLANNING.txt` + `AUTONOMOUS_RESEARCH_SYSTEM.md`
- Output contract: `OUTPUT_CONTRACT.md`

## Execution

- Follow `experiment_design.md`
- Keep outputs aligned with `result_schema.json`
- Save run data in `experiment_results.json`

## Writing

- Use `RUN_WRITING.txt` + `RESULT_TO_PAPER_ENGINE.md`
- Inputs: `research_blueprint.json`, `experiment_results.json`, optional `paper_seed.md`
- Track follow-ups in `appendix_next_runs.md`
