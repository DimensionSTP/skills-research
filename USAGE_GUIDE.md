# Usage Guide

## End-to-End Flow

1. Run planning with `AUTONOMOUS_RESEARCH_SYSTEM.md` and `RUN_PLANNING.txt`.
2. Produce required artifacts defined in `OUTPUT_CONTRACT.md`.
3. Execute experiments and fill result templates.
4. Draft paper using `RESULT_TO_PAPER_ENGINE.md` and `RUN_WRITING.txt`.

## Template Artifacts in This Repo

- `experiment_design.md`
- `result_schema.json`
- `research_blueprint.json`
- `experiment_results.json`
- `paper_seed.md`
- `appendix_next_runs.md`

These are starter templates. Replace `SAMPLE` placeholders with project-specific content.

## Validation Tips

- Keep IDs consistent across blueprint and results.
- Keep schema keys stable once runs start.
- Record assumptions explicitly if information is missing.
